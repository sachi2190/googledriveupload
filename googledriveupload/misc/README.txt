Steps to execute the project :

1. Download and open in NetBeans IDE(My version is 8.0.1).
2. Right click and do clean and build for this project.
3. Go to TestPackage-->FileUploadTestNGTest.java.
4. Right click in the file and select Run file.
5. It will lead to a browser page for allowing access to drive, Please select Allow access.
6. Copy the authorization code you got after it and paste in the Output prompt of the Netbeans as follows :

	Enter the authorization code<paste your authorization code here>
7. If the code is correct it will upload a document and an image to your google drive.
8. For verfication go to google drive and search for file document.txt and image.jpg.

DONE !!
